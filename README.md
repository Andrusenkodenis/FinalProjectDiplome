Дипломный проект по автоматизации UI-тестирования интернет-магазина, выполненный в рамках обучения.

Проект реализован на языке Java с использованием Selenium WebDriver и JUnit по паттерну Page Object Model (POM).

Автоматизированы основные пользовательские сценарии работы с веб-приложением.

Реализованные сценарии:

Авторизация:
- успешная авторизация;
- авторизация с неверным логином;
- авторизация с неверным паролем;
- авторизация с пустыми полями.

Регистрация:
- успешная регистрация;
- регистрация без имени пользователя;
- регистрация без email;
- регистрация без пароля.

Главная страница:
- переход в личный кабинет;
- поиск существующего товара;
- поиск несуществующего товара;
- переход в корзину;
- переход в раздел «Книги»;
- возврат на главную страницу по логотипу.

Каталог:
- открытие страницы каталога;
- проверка отображения товаров;
- открытие карточки товара;
- добавление товара в корзину.

Корзина:
- удаление товара;
- восстановление удалённого товара;
- применение купона.

Оформление заказа:
- переход к оформлению заказа;
- заполнение данных покупателя;
- выбор способа оплаты;
- оформление заказа.
